# 猜字谜小游戏 - 部署说明

本应用为 **纯静态单文件**，无任何外部依赖，可离线运行。部署后，任何人用手机或电脑打开链接即可使用，无需安装或配置。

## 方式一：Netlify Drop（推荐，最简单）

1. 打开 [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. 将整个 `xijing` 文件夹拖入页面
3. 几秒后会得到一个网址，如：`https://xxx.netlify.app`
4. 把网址发给朋友，用手机浏览器打开即可使用

## 方式二：GitHub Pages

1. 在 GitHub 新建一个仓库
2. 把 `index.html` 上传到仓库根目录
3. 仓库 Settings → Pages → Source 选择 main 分支
4. 保存后得到网址：`https://你的用户名.github.io/仓库名/`

## 方式三：微信/QQ 发送文件

- 将 `index.html` 发到微信/QQ
- 对方下载后，用手机浏览器打开该文件（部分安卓支持直接打开）
- 注意：iOS 上可能需借助“文件”App 或第三方浏览器打开

## 方式四：Surge（命令行一键部署）

```bash
cd /Users/xiaomao/Downloads/xijing
npx surge . --domain 自选名称.surge.sh
```

会得到类似 `https://自选名称.surge.sh` 的网址。
